##  Dataset

  
 
 ## Dataset Description

The dataset represents a telecommunications customer base and is designed for customer churn analysis. It contains 6,687 customer records with 29 attributes, covering customer demographics, account details, service usage, pricing, contracts, payments, and churn outcomes.

The data provides a holistic view of customer behavior, making it suitable for exploratory analysis, churn modeling, and decision support systems.
